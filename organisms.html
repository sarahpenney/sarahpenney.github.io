Final General Assembly Project:

Organisms is a website where you can upload essays and written work on Bioluminescent organisms.
It uses register and login to enter the site. It has a toggle switch which goes between original styling and what it would have possibly looked like in the 90's.
There is also a mobile friendly style which plays a holographic jellyfish.

Tech used: HTML, SCSS, Angular, Ruby, SQL
